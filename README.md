# ВЕЩЬ.СТОР — интернет-магазин одежды (Django)

Учебный проект: интернет-магазин по продаже одежды и аксессуаров.
Стек: Python 3.12, Django 6, SQLite, HTML/CSS/JS (серверный рендеринг шаблонов).

## Возможности
- Главная страница с hero, категориями, рекомендуемыми товарами и новинками
- Каталог с фильтром по категориям, поиском и сортировкой по цене/названию
- Карточка товара с похожими товарами и скидками
- Корзина на основе сессии (добавление, изменение количества, удаление)
- Оформление заказа с валидацией формы → заказ сохраняется в БД
- Форма обратной связи (Контакты)
- Страница «О магазине»
- Админ-панель: управление категориями, товарами, заказами и сообщениями
- Адаптивная вёрстка (десктоп / планшет / мобильный)

## Запуск (Linux / Arch)
```bash
# 1. Виртуальное окружение
python -m venv venv
source venv/bin/activate

# 2. Зависимости
pip install -r requirements.txt

# 3. Миграции (БД и демо-данные уже включены в архив — этот шаг можно пропустить)
python manage.py migrate

# 4. (опционально) заново загрузить демо-товары
python manage.py seed

# 5. Запуск сервера
python manage.py runserver
```
Открыть в браузере: http://127.0.0.1:8000/

## Структура
```
config/        — настройки проекта (settings, urls)
shop/
  models.py        — Category, Product, Order, OrderItem, ContactMessage
  views.py         — представления всех страниц
  cart.py          — корзина на сессии
  forms.py         — формы заказа и обратной связи
  admin.py         — настройка админки
  urls.py          — маршруты приложения
  context_processors.py — счётчик корзины и меню категорий
  templates/shop/  — HTML-шаблоны
  static/shop/     — CSS и JS
  management/commands/seed.py — генерация демо-данных
media/         — изображения товаров
db.sqlite3     — база с демо-данными
```
